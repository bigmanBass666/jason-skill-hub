# Skill Catalog

## 使用方法

**第一步**: 从下方选择一个 skill

**第二步**: 访问其 **Raw URL** 抓取 SKILL.md 完整内容（Description 仅供参考）

**第三步**: 如果 SKILL.md 提到其他文件（如 `./references/xxx.md`），用以下公式拼出完整 URL：

```
完整 URL = "https://raw.githubusercontent.com/bigmanBass666/jason-skill-hub/master/skills/{skill名}" + "/" + "references/xxx.md"
示例: https://raw.githubusercontent.com/bigmanBass666/jason-skill-hub/master/skills/pr-template/references/template.md
```
