# Skill Catalog

## 使用方法

**第一步**: 从下方选择一个 skill

**第二步**: 复制它的 Raw URL，访问抓取 **完整的 SKILL.md 原文**

**第三步**: SKILL.md 中如果提到其他文件，用 Raw URL 所在的目录 + 文件相对路径拼接

> **⚠️ 强制要求**：
> - 你必须实际访问 Raw URL 获取 SKILL.md 内容，**不能仅凭 description 推断**
> - 抓取后必须先输出 SKILL.md 的完整内容，证明你真的读取了
> - 必须按照 SKILL.md 里的步骤和引用文件执行任务
