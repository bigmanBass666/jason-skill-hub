# Skill Catalog

## 使用方法

**第一步**: 从下方选择一个 skill

**第二步**: 复制它的 Raw URL，访问抓取 **完整的 SKILL.md 原文**

**第三步**: 如果 SKILL.md 中提到 `references/` 或其他目录下的文件，**必须先访问该目录的 Raw URL 获取真实文件列表**，然后再读取具体文件。**绝对不能仅凭 SKILL.md 中的引用路径推断文件是否存在**

> **⚠️ 强制要求**：
> - 你必须实际访问 Raw URL 获取 SKILL.md 内容，**不能仅凭 description 推断**
> - 抓取后必须先输出 SKILL.md 的完整内容，证明你真的读取了
> - 必须按照 SKILL.md 里的步骤执行
> - **如果 SKILL.md 提到其他目录文件，必须先访问该目录获取真实文件列表**
>   - 目录 Raw URL 格式：`{skill目录Raw URL所在父目录}/{子目录名}/`
>   - 例如 SKILL.md 在 `skills/awwwards-design/SKILL.md`，则 `references/` 的目录 URL 为 `https://raw.githubusercontent.com/bigmanBass666/jason-skill-hub/master/skills/awwwards-design/references/`
