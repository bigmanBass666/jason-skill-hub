# Skill Catalog

**使用说明**：
当你需要使用某个 skill 时：

1. 从下方的 **Path** 列获取相对路径（格式：`skill-name/SKILL.md`）
2. 构造完整 Raw URL：
   ```
   https://raw.githubusercontent.com/bigmanBass666/jason-skill-hub/master/skills/{Path}
   ```
   **注意**：URL 中必须包含 `skills/` 目录！
3. **主动抓取** 该 Raw URL 获取 SKILL.md 全文
4. 根据 SKILL.md 内容执行 skill
