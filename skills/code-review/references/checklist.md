# 代码审查检查清单

## 1. 代码逻辑
- [ ] 功能是否正确实现
- [ ] 边界条件是否处理
- [ ] 错误处理是否完善
- [ ] 是否存在死代码

## 2. 代码可读性
- [ ] 命名是否清晰
- [ ] 函数是否足够小
- [ ] 注释是否必要且恰当
- [ ] 代码格式是否统一

## 3. 性能
- [ ] 是否有明显的性能问题
- [ ] 是否有不必要的循环
- [ ] 是否有内存泄漏风险

## 4. 安全
- [ ] 是否有注入风险
- [ ] 是否验证了用户输入
- [ ] 是否暴露敏感信息

## 5. 测试
- [ ] 是否有适当的测试
- [ ] 测试是否覆盖边界情况
