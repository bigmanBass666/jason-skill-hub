# PR Template Skill

## 概述
本 skill 用于生成标准化的 Pull Request 描述模板，确保每个 PR 都包含必要的信息。

## 使用时机
- 当准备提交 PR 时
- 当需要生成变更说明时
- 当需要记录测试验证步骤时

## 步骤
1. 收集变更相关信息
2. 使用 ./references/template.md 生成 PR 描述
3. 根据具体项目调整模板内容
